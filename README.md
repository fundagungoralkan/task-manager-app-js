<h1>Task Manager App (JavaScript)</h1>

<p><strong>Task Manager App</strong> is a lightweight, responsive, and user-friendly web application for managing daily tasks and notes efficiently. Built entirely with <strong>HTML, CSS, and JavaScript</strong>, this app allows users to create, edit, and delete tasks in an intuitive interface.</p>


<h2>Features</h2>
<ul>
  <li><strong>Add New Notes</strong> – Quickly create new tasks with a title and description.</li>
  <li><strong>Edit & Delete Tasks</strong> – Update or remove tasks easily.</li>
  <li><strong>Responsive Design</strong> – Works smoothly on desktop, tablet, and mobile devices.</li>
  <li><strong>Interactive UI</strong> – Popup form for adding tasks and animated status messages.</li>
  <li><strong>Lightweight & Fast</strong> – Minimal dependencies, fully client-side.</li>
</ul>

<h2>Technologies Used</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript (ES6)</li>
  <li>Boxicons</li>
</ul>

<ol>
  <li><strong>Clone the repository</strong>
    <pre>git clone https://github.com/your-username/task-manager-app-js.git</pre>
  </li>
  <li><strong>Navigate to the project folder</strong>
    <pre>cd task-manager-app-js</pre>
  </li>
  <li><strong>Open <code>index.html</code> in your browser</strong></li>
</ol>
